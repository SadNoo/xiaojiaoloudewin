## 当前建议基线

- Windows 10/11 x64，每用户安装。
- Python 业务内核 + React 前端 + WebView2 桌面壳。
- 在线服务部署在 Cloud 或自有 VPS，首版负责授权码、设备绑定、远程吊销、离线票据和版本策略。
- 云端包含所有者页面和受权限/额度限制的个人子管理员页面，不包含销售、支付或代理商结算。
- 授权中心部署在 Debian VPS，通过管理专用二级域名提供 HTTPS 服务。
- 授权支持永久、按天数到期和按自然月到期；每个子管理员默认最多维护 100 个有效授权码。
- 闲鱼 Cookie、订单、聊天、卡券和 AI Key 默认仅存本机并加密。
- 首次激活必须联网；每次启动优先联网验证。仅服务器不可达时允许使用离线宽限，明确吊销不得离线放行。
- 授权采用一机一码、默认不限制闲鱼账号数量、72 小时离线宽限和 15 分钟授权心跳；到期后停止自动任务，但保留只读和数据导出。
- Windows 客户端采用 Python + React + WebView2，支持 Windows 10/11 x64，每用户安装，默认托盘运行且不开机自启。
- 首版优先保留多账号、扫码登录、回复、AI、订单、卡券和自动发货。

核心授权与 Windows 客户端默认参数已经冻结；实际二级域名、VPS 规格、品牌信息、子管理员总数和 P0 业务功能细项仍需在开发/部署前填写。
