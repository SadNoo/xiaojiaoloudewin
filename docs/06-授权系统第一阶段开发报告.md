# 授权系统第一阶段开发报告

> 完成日期：2026-07-11  
> 代码目录：`license_server/`  
> 阶段状态：最小闭环已实现并通过自动化测试，尚未部署到 VPS。

## 已完成

- 独立 FastAPI 授权服务，不改动原闲鱼业务代码。
- SQLite 本地开发与 PostgreSQL 生产配置。
- 所有者初始化、后台登录、个人子管理员创建和额度隔离。
- 永久、按天、按自然月授权码生成。
- 授权码不以可直接使用的明文写入数据库。
- 一机一码激活、第二台设备拒绝、刷新令牌轮换。
- 15 分钟心跳、授权和单设备远程吊销。
- 每 30 天最多 2 次客户端自助解绑。
- Ed25519 短期访问票据、72 小时离线票据和公开验证公钥。
- 最低版本、禁止版本、强制更新和签名版本清单。
- 审计事件数据结构和关键操作记录。
- Dockerfile、PostgreSQL、Caddy 和 Docker Compose 部署基础。
